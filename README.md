# my_shortcuts

## PyCharm


### File Navigation
Go to specific line number 
  - Windows: ```ctrl + g```
  - Mac: ```command + l```

Move cursor to next line ```ctrl + shift + enter```
 
### Text Editing
Delete current line 
  - Windows: ```ctrl + backspace``` 
  - Mac: ```command + backspace```

Multiple Cursors
  - Mac: ``` Double option + up/down arrows```

### Execution 
Run current file ```control + shift + R```
